# Amazon-Automacao-de-Busca
Este projeto é um script em Python que automatiza a busca de produtos no site da Amazon Brasil, coleta nome e preço dos produtos encontrados e salva os resultados em um arquivo Excel.

Funcionalidades

✅ Leitura de produtos a partir de arquivo Excel
✅ Acesso automatizado ao site da Amazon Brasil
✅ Busca automática por cada produto
✅ Extração do nome do produto
✅ Extração do preço do produto (com fallback de formato)
✅ Ignora anúncios e resultados inválidos
✅ Salva os resultados em um novo arquivo Excel
✅ Uso de WebDriverWait para evitar erros de carregamento

Tecnologias Utilizadas

Python 3
Selenium
Pandas
Google Chrome / ChromeDriver

Objetivo do Projeto
Este projeto foi desenvolvido com foco em:
aprendizado de Selenium
automação web
tratamento de elementos dinâmicos
boas práticas com pandas + Excel
validação de dados coletados
